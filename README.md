# Pushpam Ayurveda Website

Static website prepared for GitHub Pages preview.

## GitHub Pages Preview

1. Create a new GitHub repository.
2. Upload or push all files from this folder.
3. In GitHub, open **Settings > Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the `main` branch and `/root`, then save.
6. GitHub will publish the preview URL in a few minutes.

The homepage entry file is `index.html`.
